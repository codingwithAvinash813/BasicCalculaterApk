# Basic Calculator App

A simple calculator Android application that performs basic arithmetic operations.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- User-friendly interface

## Installation
1. Download the APK from the [Releases](https://drive.google.com/file/d/1DgmnXZip95151RBFfkebbfec0CXRlaxg/view?usp=drivesdk) section.
2. Install the APK on your Android device.
3. Open the app and start calculating!

## How to Use
1. Enter numbers using the on-screen keypad.
2. Choose an arithmetic operation (+, -, *, /).
3. Press `=` to see the result.
4. Use `C` to clear the input.
